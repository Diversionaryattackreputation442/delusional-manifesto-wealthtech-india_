## 📚 Understanding Terms in This Manifesto

The manifesto uses some finance and tech terms. Here are clear meanings for a few key ones:

- **Account Aggregator:** A system that collects financial information from multiple sources with your permission.
- **Agentic AI:** Artificial intelligence that can act independently, making decisions or taking actions without direct input.
- **India Stack:** A set of APIs and digital infrastructure that supports government-backed digital services in India.
- **Fintech:** Technology used to improve and automate financial services.
- **Wealthtech:** Technology focused on managing personal wealth and investments.
